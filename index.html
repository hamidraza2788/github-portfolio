School ERP Mobile Application

Multi-Tenant Educational Management Platform

A comprehensive, enterprise-level mobile application built with React Native (Expo) that bridges the gap between educational institutions, parents, students, and service providers. The platform features a sophisticated role-based architecture supporting over 8 distinct user types, each with tailored dashboards and functional suites.

Key Features & Modules

  • Multi-Role Ecosystem: Dynamic role-based authentication supporting School Admins, Teachers, Parents/Students, Transporters, Micro-Financers, and Vendors with tailored functional flows.
  • Academic Management: Performance tracking with graphical insights, interactive attendance calendars, digitized exam schedules, and real-time homework/notice board updates.
  • Financial Operations: Digital fee collection with history tracking, integrated wallet system, and unique micro-finance loan management for staff and parents.
  • Vendor Marketplace: Dedicated portals for stationery and uniform sellers to manage inventories and process school orders.
  • Communication Hub: Real-time chat system, multimedia gallery for school events, and automated push notifications for critical updates.
  • Advanced Architecture: Modular routing with dynamic navigation, multiple Axios instances for auth and API calls, Redux Toolkit state management, and full TypeScript implementation.

Technical Stack

React Native (Expo SDK 52) TypeScript Redux Toolkit Redux Persist React Navigation React Hook Form Axios Moment.js Crypto-js React Native Reanimated

Technical Highlights

  • Modular Routing: Dynamic BottomTabNavigator that adapts based on user role with createStaticNavigation architecture.
  • Advanced API Layer: Specialized Axios instances (erpAuthInstance, erpAxiosInstance) with JWT injection via Redux-linked interceptors.
  • State-Driven UI: Redux Toolkit Slices for global state management (auth, schoolDetail, loans, enrollments) ensuring data consistency across 250+ reusable components.
  • Performance Optimized: Lazy loading, memoized components, and full TypeScript implementation for enterprise-level scalability.
